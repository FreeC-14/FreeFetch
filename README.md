# Simple Fetch

Python ile yazılmış minimal sistem bilgi gösterici.

---

## Özellikler

- Aktif kullanıcı adı  
- Bilgisayar adı (hostname)  
- İşletim sistemi ve çekirdek sürümü  
- Sistem çalışma süresi (dakika)  
- ASCII kedi resmi

---

## Gereksinimler

- Python 3.x  
- Linux (`/proc/uptime` dosyasını okur)

---

## Kullanım

```bash
python3 fetch.py
